== Description ==

[Rot13 Email Protection](http://techtrouts.com/rot13-email-protection-for-wordpress/ "Rot13 Email Protection for Wordpress") for Wordpress, protects all email addresses and mailto: links on your blog from email harvesters using rot13 encryption and/or text mode. By [9tree](http://9tree.net "9tree, Natural Revolution for the Web").

== Installation ==

1. Upload `rot13_email_protection.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Edit your settings under the submenu 'Rot13 Email Protection' on the 'Plugins' page.
